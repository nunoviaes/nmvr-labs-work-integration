# nmvr-labs-work-integration-templates

ARM Templates playground for multiple Azure Services using CI / CD pipelines in Azure DevOps. 

This includes: 

* Service Bus
* Event Grid
* Event Hubs
* Logic Apps
* Logic Apps (Preview)
* Azure Functions
* API Management
* Cosmos DB
* SQL Database
* Others
